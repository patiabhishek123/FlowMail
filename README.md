# FlowMail

FlowMail is a powerful email management application featuring an intuitive React frontend and a robust Go backend. This project aims to provide users with a seamless experience for managing email spam campaigns, similar to MailChimp, while prioritizing design simplicity and performance.

## Features

- **React Frontend**: The frontend of FlowMail is built with React, ensuring a fast and interactive user experience. The minimalist design allows users to navigate through functionalities effortlessly.

- **Go Backend**: The backend is powered by Go, known for its concurrency capabilities, which makes it perfect for handling numerous requests simultaneously. FlowMail leverages Go’s performance to ensure quick response times and efficient processing.

- **Email Spam Campaign Dispatcher**: Inspired by MailChimp, FlowMail includes a comprehensive spam campaign dispatcher which allows users to create, manage, and analyze email campaigns effectively.

- **Concurrency Model Testing**: FlowMail implements a rigorous concurrency model that has been tested for performance and reliability, ensuring that it can handle large volumes of email dispatches without a hitch.

- **Resend API Integration for SMTP**: To facilitate reliable email dispatch, FlowMail integrates the Resend API, allowing for seamless SMTP operations. This integration helps to maintain high delivery rates and manage bounced emails effectively.
